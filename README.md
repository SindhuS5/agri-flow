# AgriFlow: Smart Agriculture Ecosystem
**Primary Theme:** Agri-Tech & Rural Development

## 🚀 Overview
AgriFlow is a digital solution for farmers in rural India (specifically Andhra Pradesh) to track crop cycles, manage finances, and find cold storage.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Scikit-learn, Streamlit, GeoPandas
- **UI:** Multilingual Support (Telugu, Hindi, English)

## 🌟 Key Features
- **Digital Ledger:** Real-time expense and profit tracking.
- **Cold Storage Mapping:** GPS-based directory for harvest safety.
- **Inclusive Design:** Voice-to-text and icon-based navigation for all literacy levels.

## 📈 Impact
Aims to increase rural farmer income by 25% through waste reduction and data-driven planning.
